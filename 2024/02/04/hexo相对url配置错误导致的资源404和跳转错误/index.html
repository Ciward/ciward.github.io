hexo相对url配置错误导致的资源404和跳转错误

问题描述

点击导航栏首页跳转至网站根目录也就是首页，头像不知道为什么没链接成功，多处资源404

1.只跳转到了当前的根目录

2.切换目录导致头像图片没链接成功

解决方案

很简单，hexo 配置 relative_link : true 即可（默认是true的，可能是折腾图片显示时改的，因为我就是😭）
